PdFiles

PdFiles.PdFiles — Module
PdFiles

Implementations of a better Filehandles.

Includes a Read-Only Filehandle, a better Write-Only Filehandle as well as Mmap-Filehandles allowing both Reading and writing (but not appending (yet)).

Main Functions: pdopen and @om_str

Open a file via Mmap in Read-Write Mode.

Appending is currently not possible and will throw an EOFError.

PdLibc

Interface for all the C-Calls and Constants used.
